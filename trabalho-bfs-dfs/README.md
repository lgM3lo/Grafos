# Trabalho BFS e DFS - Região Nordeste

## 📌 Descrição

Este projeto implementa um grafo que representa os estados da região Nordeste do Brasil, onde:

- Cada vértice representa um estado
- Cada aresta representa uma fronteira terrestre entre estados

Foram utilizados os algoritmos:

- Busca em Profundidade (DFS)
- Busca em Largura (BFS)

seguindo o padrão da biblioteca algs4.

---

## 🗺️ Modelagem do Grafo

Estados (ordem alfabética):

0: AL  
1: BA  
2: CE  
3: MA  
4: PB  
5: PE  
6: PI  
7: RN  
8: SE  

As arestas foram definidas com base nas fronteiras terrestres reais entre os estados.
